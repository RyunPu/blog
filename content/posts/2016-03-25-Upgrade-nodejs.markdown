### Use NPM

```bash
$ sudo npm cache clean -f
```

```bash
$ sudo npm install n -g
```

```bash
$ sudo n stable
```

or

```bash
$ sudo n 8.0.0
```

### Use Homebrew

```bash
$ brew upgrade node
```

### Use NVM

```bash
$ nvm install 8.0.0
```

```bash
$ nvm use 8.0.0
```
